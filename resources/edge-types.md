# Edge Types

**Common Edge Types**

|  | Number of Planes | Angle | Parallel to Ground |
| :--- | :--- | :--- | :--- |
| Eave | 1 |  | √ |
| Rake | 1 |  | × |
| Hip | 2 | "Λ" | × |
| Valley | 2 | "V" | × |
| Ridge | 2 |  | √ |
| Flashing | Only around the chimney |  |  |
| Step Flashing | Only connects towards the wall |  |  |
| Parapet | Only on the flat roof that connects towards the wall |  |  |
